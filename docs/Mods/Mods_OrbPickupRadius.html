Orb Pickup Radius

Widen orb auto-magnet

A data-only patch mod that widens the range at which you can pick up orbs (XP, credits, health, endurance)

Install

Copy the patch JSON patch file into your server's patch folder:

from repo patches/PatchDataPickupRadiusOrbs.json into your built server patches folder , for example = 
MHServerEmu/Data/Game/Patches/
MHServerEmu\bin\x64\Release\net8.0\Data\Game\Patches

Restart the server. Make sure [GameData] EnablePatchManager=true in Config.ini.
